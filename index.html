Hypertext Markup Language, as the name implies,
HTML is not a programming language,
it is a markup language, which consists of grouping
or delimiting parts of the content we create to perform a certain function.
We use titles, paraphrases, images, assembling part by part.
Example:

< p>Hello Word< p/>

Where "p" is our paragraph, to which we can add classes, text types,
among many other things.
And we have the help of CSS, to "embellish" our site, because we can use it to
style texts, add things (boxes, logos, sparkles, fonts, etc...).

Cascading Style Sheet,
is a style language used to describe
the presentation of a document written in HTML or XML
(including various XML languages such as SVG, MathML or XHTML).
It shows how elements are displayed on the screen,
with images, text styles, audios, among other media.
